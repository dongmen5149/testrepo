Hero4 는 **소환수(환수) 시스템** 보유. `_H_SS` = 환수 catalog (5+ 환수 × 각 5 스킬 + 4 글로벌 강화).

## 환수당 5 스킬 (반복 구조)

각 환수마다 동일 layout:

1. `{환수}의 기본공격력 강화` (기본공격 뇌격)
2. `{환수}의 원거리 뇌격 강화` (원거리)
3. 뇌격의 {속성} 효과 강화 (저주/슬로우/스턴/회복)
4. `{속성}의 오러` (저주/강화/마법/마력/보호)
5. 소환시 소환자의 {버프} (저주증가/저항증가/회복증가/방어력증가)

→ 총 25 환수 스킬 (5 환수 × 5 스킬).

## 4 글로벌 강화 (모든 환수 공통)

마지막 4 entries:
- 마법력강화 — 소환수의 마법력 강화
- 교감도강화 — 소환수의 방어력 강화
- 체력강화 — 소환수의 체력 강화
- 정신강화 — 소환수의 교감도 강화

→ **소환사 (Summoner)** 의 패시브 강화 스킬 4종.

## 추정 캐릭터 mode 연결

R81 의 "2 영웅 × 2 mode" + R84 enchantment 의 클래스 보완 affix → 환수 시스템은 **3번째 캐릭터/클래스 가능성**:
- 티르 (양손검 / 마검)
- 루레인 (사격 / 단도+마법)
- **소환사** (환수 5종) ← 신규 발견

또는 루레인 mode 1 (S003 단도+마법) 의 마법 부분이 환수 소환을 포함할 가능성. S003 스킬 "환수흡수" 가 직접 매칭.
