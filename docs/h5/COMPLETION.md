## 4. 지표 C — 베타 서비스 오픈 (~50%, 원작 동등 플레이 = 100%)

**100% =** 신규 유저가 타이틀→클래스→**본편 첫 마을/오프닝**부터, 퀘스트·NPC·워프·맵 조우·전투·상점·세이브가 **원작과 같은 화면·순서**로 끊김 없이 진행 (M/N·B·G 등 **개발용 키 불필요**).

### 4.0 왜 종합 87.6%와 크게 다른가

| 구분 | 포함 여부 | R108 체감 |
|------|-----------|-----------|
| .so RE, JSON, Formula VM | 종합 **포함** | 95–96% |
| 10개 UI 패널·전투·AI (단독 검증) | 종합 **포함** | 90%+ |
| **Title→ClassSelect→`demo.tscn` 본편 연결** | 베타 **핵심** | **~20%** — New Game 후 **통합 데모 샌드박스** |
| **258 `.scn` 시나리오 자동 진행** | 베타 **핵심** | **~40%** — 씬 로드 시 body **최대 64 step**, 다수 opcode **print/stub** |
| **맵 조우→전투→복귀** (원작 Event 경로) | 베타 **핵심** | **~50%** — B/G·필드 AI는 있으나 **시나리오 Event_Encount 연동 미완** |

→ RE·패널·자동테스트가 높아 **종합은 8할 후반**, **원작 플레이 동등성은 5할** — **둘 다 맞고, 정의만 다름**.
